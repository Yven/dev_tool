项目需求说明书（v0.1）

一、产品定位  
开发一款 Windows 桌面轻量级辅助工具，具备“秒级启动、极低内存、插件式扩展”三大核心特征，面向开发者及运维人员日常高频小需求场景。

二、功能性与非功能性需求  
1. 启动性能  
   - 冷启动 ≤ 800 ms（机械硬盘实测）  
   - 热启动 ≤ 300 ms  
   - 内存占用 ≤ 35 MB（仅加载核心框架，无插件时）  

2. 交互模型  
   - 全局快捷键：默认 Ctrl+Shift+Space，可自定义；注册系统级热键，保证在任意窗口层级可唤出  
   - 贴边隐藏：支持屏幕上、左、右三边吸附；鼠标移入吸附区域 120 ms 内完整展开窗口，移出 400 ms 后自动缩回；缩回时仅保留 3 px 触发条  
   - 多屏适配：正确识别虚拟屏坐标，防止窗口超出可视区域  

3. 核心功能（v0.1 内置 4 个官方模块，后续通过插件扩展）  
   ① 时间戳转换  
      - 支持秒/毫秒/微秒级互转  
      - 支持 UTC 与本地时区双向切换  
      - 支持常见格式（RFC3339、ISO8601、Unix、Excel）一键复制  
   ② UTF-8 中文转换  
      - 输入框实时检测编码，自动识别 \uXXXX、&#x、%E 等转义  
      - 提供“编码→解码”“解码→编码”双向按钮  
   ③ URL 编码/解码  
      - 符合 RFC 3986，保留字符白名单可配置  
      - 支持批量处理多行文本  
   ④ JSON 格式化  
      - 解析失败时给出行号及错误原因  
      - 支持压缩、转义 Unicode、排序键名、复制纯文本/复制带缩进两种模式  

4. 插件体系（为 v0.2 预留接口，v0.1 需完成底层加载器）  
   - 采用 ESModule 规范，插件单文件 ≤ 200 KB  
   - 插件隔离沙箱，禁止直接访问 Node API，仅暴露官方 SDK  
   - 热插拔：插件放入 plugins 目录立即生效，删除即卸载，无需重启主程序  

5. 系统兼容  
   - 最低支持 Windows 10 1903（Build 18362）  
   - 打包为 64 位可执行文件，附带无人值守静默安装参数 /S  
   - 安装包 ≤ 8 MB，首次运行不产生除 logs 外的额外文件  

三、项目里程碑  
Phase 1 项目启动  
   - 输出《项目结构设计方案》：目录分层、技术选型（Electron / Tauri / C# 对比表）、构建流程、CI/CD 脚本  
   - 输出《项目立项说明书》：目标市场、竞品对比、ROI 评估、风险列表、人员角色、预算与排期  

Phase 2 核心原型  
   - 完成主进程与渲染进程骨架，实现插件加载器（空转）  
   - 完成全局快捷键、贴边隐藏、窗口阴影、多屏坐标计算  
   - 单元测试覆盖率 ≥ 70%，集成测试用例 ≥ 30 条  

Phase 3 功能交付  
   - 四个内置模块全部完成并通过验收测试  
   - 性能基线达标：启动时间、内存、CPU 占用通过自动化脚本每日回归  
   - 发布 v0.1 安装包与绿色版，附带使用手册与快捷键说明  

Phase 4 复盘与迭代  
   - 收集 100+ 真实用户日志，输出《v0.2 需求优先级列表》  
   - 完成 Agents（自动化构建、测试、发布、崩溃上报、更新推送）脚本固化到仓库 .github/workflows
